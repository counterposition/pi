# ADRs

This directory holds monorepo-level architectural decision records.

Use `docs/adrs/` for decisions that affect the repository as a whole, such as workspace layout, release flow, shared tooling, or cross-package conventions.

Use `packages/<name>/docs/adrs/` for decisions that are scoped to one package.
